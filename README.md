## Quick Start
1. Create a new repo from this template.  
2. Open the repo in **GitHub Codespaces**.  
3. Enable **Copilot Agent Mode**.  
4. Tell Copilot:  
   > "Read `COPILOT_INSTRUCTIONS.md` and help me start building my app."
